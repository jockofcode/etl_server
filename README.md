ETL Server is the frontend for the ETL Engine in ../etl_engine.

After a user is logged into the ETL Server UI, they are presented with a searchable list of all the flows the user has access to. Both ones they created and ones that have been shared with them. When they click on the flow, they are taken to the flow view that shows the whole chain of operations in the flow, and when they click on an operation, a card pops up with all the details of that operation, giving the user the option to edit or remove or close it. In the flow view the user has the option to run the flow.
